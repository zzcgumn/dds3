This is to build and work ont a macos system.
## Apple tools
Xcode's command line tools. 

## Build system and compiler
```
brew install bazelisk
brew install llvm
```

## MCP servers
Install [go lang environment](https://go.dev/doc/install)
Clone [serena](https://github.com/oraios/serena) under $HOME/Source/Python
```
brew install node
go install github.com/isaacphi/mcp-language-server@latest
```
